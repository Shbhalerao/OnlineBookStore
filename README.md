<h1 style="text-align: center; color: #4A90E2;">Online Book Store</h1>

<p style="font-size: 18px; text-align: center;">Welcome to my repository! This project showcases an online bookstore created using <strong>Java</strong>, <strong>HTML</strong>, <strong>CSS</strong>, <strong>JSP</strong>, <strong>Servlets</strong>, and <strong>MySQL</strong>. The application allows users to browse, search, and purchase books online, providing a user-friendly interface and secure transactions.</p>

<hr>

<h2 style="color: #4A90E2;">Features</h2>
<ul style="list-style-type: square; padding-left: 20px;">
    <li>User registration and authentication</li>
    <li>Browse and search for books</li>
    <li>Add books to the shopping cart</li>
    <li>Secure checkout process</li>
    <li>Admin panel for managing inventory</li>
</ul>

<h2 style="color: #4A90E2;">Technologies Used</h2>
